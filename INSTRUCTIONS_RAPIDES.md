# Instructions RAPIDES et SÛRES

## Méthode 1 : Script Batch (30 secondes)

1. **Double-cliquez sur le fichier `restore_safe.bat`** dans votre dossier
2. Le script va :
   - Créer une branche de sauvegarde automatiquement
   - Restaurer le commit dsfbdb
   - Vos fichiers seront mis à jour dans Cursor automatiquement

**Pour revenir en arrière :** Dans GitHub Desktop, basculez sur la branche "sauvegarde-avant-restore"

---

## Méthode 2 : GitHub Desktop (1 minute) - LA PLUS SÛRE

1. **Ouvrez GitHub Desktop**
2. **History** (onglet en haut)
3. **Trouvez "dsfbdb"** dans la liste
4. **Clic droit** → **"Create branch from commit"**
5. **Nommez-la** : `version-dsfbdb`
6. **C'est tout !** Cursor se met à jour automatiquement

**Avantage :** Votre branche main reste intacte, vous pouvez revenir dessus à tout moment !
