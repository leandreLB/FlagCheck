# Guide ULTRA SIMPLE - GitHub Desktop

## En 4 étapes (2 minutes) :

### Étape 1 : Ouvrez GitHub Desktop
- Lancez l'application GitHub Desktop

### Étape 2 : Cliquez sur "History" 
- C'est l'onglet en haut à gauche, à côté de "Changes"

### Étape 3 : Trouvez "dsfbdb"
- Faites défiler la liste des commits
- Cherchez celui qui dit "dsfbdb" (il est à environ 3 heures dans le passé)

### Étape 4 : Clic droit → "Create branch from commit"
- **Clic droit** sur le commit "dsfbdb"
- Dans le menu, cliquez sur **"Create branch from commit"** ou **"Créer une branche à partir de ce commit"**
- Nommez la branche : `version-dsfbdb` (ou n'importe quel nom)
- Cliquez sur "Create branch"

## Résultat :
- GitHub Desktop va créer la branche et basculer dessus
- **Cursor va automatiquement voir les changements** et mettre à jour les fichiers
- Votre branche main reste intacte !

## Pour revenir à votre version actuelle :
- Dans GitHub Desktop, cliquez sur "Current branch" en haut
- Sélectionnez "main"
- Vos fichiers reviennent à la version actuelle !

**C'est vraiment la méthode la plus simple et sûre !**
