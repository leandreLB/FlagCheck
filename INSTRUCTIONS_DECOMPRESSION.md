# Instructions : Où mettre le fichier ZIP

## Étapes détaillées :

### 1. Téléchargez le ZIP
- Le fichier s'appelle : `FlagCheck-988734494c81f6c8d4cd6b89285c93862f261ba4.zip`
- Il est probablement dans votre dossier **Téléchargements**

### 2. Décompressez le ZIP
- **Double-cliquez** sur le fichier ZIP
- Windows va extraire les fichiers dans un nouveau dossier qui s'appelle : `FlagCheck-988734494c81f6c8d4cd6b89285c93862f261ba4`

### 3. Où mettre le dossier décompressé ?

**Option A : Garder dans Téléchargements (RECOMMANDÉ pour tester)**
- Laissez-le où il est (dans Téléchargements)
- Ouvrez ce dossier dans Cursor

**Option B : Le déplacer à côté de votre projet actuel**
- Coupez (Ctrl+X) le dossier décompressé
- Allez dans `C:\Users\Léandre\`
- Collez (Ctrl+V) le dossier là
- Vous aurez :
  - `flagcheck` (votre projet actuel)
  - `FlagCheck-988734494c81f6c8d4cd6b89285c93862f261ba4` (l'ancienne version)

### 4. Ouvrir dans Cursor

**Dans Cursor :**

1. **Fermez le dossier actuel** :
   - Menu `File` → `Close Folder`
   - Ou appuyez sur `Ctrl+K` puis `Ctrl+F`

2. **Ouvrez le nouveau dossier** :
   - Menu `File` → `Open Folder...`
   - Naviguez vers le dossier décompressé :
     - Si dans Téléchargements : `C:\Users\Léandre\Downloads\FlagCheck-988734494c81f6c8d4cd6b89285c93862f261ba4`
     - Si déplacé : `C:\Users\Léandre\FlagCheck-988734494c81f6c8d4cd6b89285c93862f261ba4`
   - Sélectionnez le dossier et cliquez sur "Sélectionner le dossier"

**Et voilà !** Vous avez maintenant l'ancienne version du code dans Cursor.

---

## Si vous voulez remplacer complètement votre projet actuel :

⚠️ **ATTENTION : Cela supprimera tous vos changements actuels !**

1. **Fermez Cursor**

2. **Renommez votre dossier actuel** :
   - Allez dans `C:\Users\Léandre\`
   - Renommez `flagcheck` en `flagcheck-backup` (pour sauvegarder)

3. **Renommez le dossier décompressé** :
   - Renommez `FlagCheck-988734494c81f6c8d4cd6b89285c93862f261ba4` en `flagcheck`

4. **Rouvrez Cursor et ouvrez le dossier `flagcheck`**

---

**Je recommande l'Option A (garder dans Téléchargements) pour commencer, comme ça vous pouvez comparer les deux versions !**
